OpenSplice DDS

Documentation Set

The Documentation Set is a collection of PDF and HTML documents which provides the information needed to install and use OpenSplice DDS. The Documentation Set currently includes:

  • Release Notes - provides the latest information about the current OpenSplice DDS release, including:
    • the latest features and technical information
    • supported platforms and compilers
    • bugs that have been fixed
    • known issues
  • Getting Started Guide (pdf) - the document that users should read first. The Getting Started Guide provides:
    • information for running your first example
    • basic installation and configuration information
    • how to license your installation
  • OpenSplice DDS DCPS PreProcessor User Manual (pdf) - provides detailed technical information on using and running the OpenSplice DDS IDL PreProcssor.
  • Deployment Guide (pdf) - information on configuring the OpenSplice DDS services.
  • DCPS C Reference Guide (pdf) - a detailed reference describing the OpenSplice DDS DCPS Application Programmers Interface (API) for the C Language.
  • DCPS C++ Reference Guide (pdf) - a detailed reference describing the OpenSplice DDS DCPS Application Programmers Interface (API) for the C++ Language.
  • DCPS Java Reference Guide (pdf) - a detailed reference describing the OpenSplice DDS DCPS Application Programmers Interface (API) for the Java Language.
  • DCPS C Tutorial (pdf) - training for using the C language binding of the OpenSplice DDS DCPS API.
  • V2-V3 Migration Guide (pdf) - guide to assist OpenSplice V2 users to migrate to OpenSplice DDS V3.

OpenSplice DDS Directory Organisation

The OpenSplice DDS components are located in the following sub-directories of the installation:

  • bin - contains OpenSplice DDS executables.
  • custom_lib - contains source code of the Corba C++ language binding, for the customer to built.
  • docs - contains the documentation set and release notes for OpenSplice DDS.
  • etc - the default location for the license file and configuration file(s). In case of the HDE, the templates for code generation are also located in this directory.
  • examples - contains code examples (for many programming languages) demonstrating the use of OpenSplice DDS.
  • include - contains all OpenSplice DDS header files.
  • jar - contains all JAVA archives for OpenSplice DDS API's and OpenSplice DDS tools.
  • lib - contains all OpenSplice DDS library files.
  • redist - depending on the platform this directory contains redistributable packages needed by OpenSplice DDS. These packages are automatically installed.
  • release.com - running this script in your current shell will setup the OpenSplice DDS environment. (only applicable to unix systems)
